# Model Weights

Put the required model checkpoints in this folder before running inference.

Required files:

```text
groundingdino_swint_ogc.pth
sam_vit_b_01ec64.pth
```

Download links:

- `groundingdino_swint_ogc.pth`: <https://github.com/IDEA-Research/GroundingDINO/releases/download/v0.1.0-alpha/groundingdino_swint_ogc.pth>
- `sam_vit_b_01ec64.pth`: <https://dl.fbaipublicfiles.com/segment_anything/sam_vit_b_01ec64.pth>

These files are intentionally not committed to GitHub because they are large model binaries.
